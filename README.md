👵 SENIOR_MOTION_DETECTION 👴
========================
노령층 이상행동 감지 및 보호자 알림 서비스


🌳 프로젝트 개요
=======================
65세 이상의 고령 인구가 증가함에 따라, 홀로 계시는 어르신 또한 많아지고 있다.

이 프로젝트는 혼자 계시는 어르신의 행동 특성과 발생 소리를 모니터링하여 

위험사항을 예측 및 감지하고 보호자에게 즉시 알리기 위해 제작되었다.
